# 运行此示例

## -1- 启动可流式传输的 HTTP 服务器

从该项目的根目录打开一个终端并运行以下命令：

```bash
cd ../calculator-httpserver
cargo run
```

保持此终端打开，因为它将运行客户端需要连接的 HTTP 服务器。接下来的步骤请打开一个新的终端。

## -2- 安装依赖项并构建项目

```bash
cargo build
```

## -3- 运行示例

```bash
cargo run
```
