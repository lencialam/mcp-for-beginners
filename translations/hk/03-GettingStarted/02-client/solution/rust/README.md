# 執行此範例

> [!NOTE]  
> 此 Rust 客戶端假設你已安裝 dotnet 8.0 SDK，用於執行 [server](../../../../../../03-GettingStarted/02-client/solution/server) 目錄中的範例 MCP 伺服器，該目錄是此專案的同級目錄。

## -1- 安裝依賴項並建置專案

```bash
cargo build
```

## -2- 執行範例

```bash
cargo run
```
